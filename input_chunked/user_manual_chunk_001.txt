上下文：雨燕租车经理人是网约车租赁中介的数字化管理工具，提供全流程线上服务，助力企业提效降本。注册需通过微信小程序完成实名或企业认证，提交身份证、营业执照等材料，当前免费开放注册。

一、雨燕经理人简介
1.1 什么是雨燕租车经理人
雨燕租车经理人是雨燕租车平台专为网约车租赁中介企业打造的销售管理工具。从“获客 → 销售 → 签约 → 验车 → 培训”全流程数字化，便于中介企业提升业务效率，降低人工成本，扩大经营范围，获取更大利润。

1.2 为什么要成为雨燕租车经理人
作为雨燕平台认证的经理人，你将能够：

获得平台提供的各类车辆资源（新能源、油车、网约车等）

在线完成租车闭环：包括推荐方案、签约、支付、交车、退车等

自动获取平台推荐的潜在客户线索，无需高成本寻找司机客户

二、如何成为雨燕租车经理人
2.1 雨燕经理人注册流程
【适用对象：经理人（个人/企业）】
小程序入口方式：
微信搜索：“雨燕租车”小程序

微信扫码：扫描平台二维码直接进入注册页面

注册与认证材料准备：
点击“立即加入”按钮 → 进入实名认证/企业认证页面

根据身份选择：

个人经理人认证：适用于个体，需提交身份证、银行卡等

企业经理人认证：适用于中介公司，需提交法人信息、营业执照等

所需材料（按认证类型区分）：

居民身份证（正反面，清晰）

个体工商户或公司营业执照

银行开户信息（法人或对公账户）

照片示例：所有证件需清晰、有效期内

实名认证流程（个人经理人）：
上传身份证正反面图片

如果是由其他经理人推荐，则填写推荐码（用于佣金绑定）

企业认证流程（企业经理人）：
上传法人身份证（正反面）

选择营业执照类型（企业/个体户）并上传清晰营业执照图片

填写公司账户信息

如果没有对公账户，可填法人个人银行卡账户

如果支行名称搜索不到，可选择所属总行

企业会员费

当前阶段免费开放注册（建议尽快认证）

银联认证流程（经理人企业）：
注：该认证流程适用于企业经理人，需通过银商平台进行实名审核。

所需材料包括：

身份证正反

营业执照（三证合一）

对公账户信息（含银行开户许可证/印章页）

法人及股东资料（身份信息）

雨燕小程序截图、门头照片、店内照片（需加水印）

步骤：

进入“银商商户自助签约平台”（PC推荐）

填写身份信息、商户信息、银行账户信息

完成小额打款验证（确认收到金额）

上传相关图片与门店信息

签署电子协议（法人手机号验证码）

等待银联审核

2.2 雨燕经理人功能介绍
2.2.1 当经理人未绑定企业或未完成企业认证时：
经理人行为： 可以分享租车方案给司机

司机体验： 可以点击查看，但无法下单或签约

解决方案： 建议经理人尽快绑定企业或完成企业认证

2.2.2 经理人加入已有企业
方式一：经理人自主绑定企业
【操作对象：经理人】

路径：
我的 → 绑定企业 → 搜索企业名称 → 提交申请 → 等待审核

审核流程：
企业管理员登录 → 企业管理 → 经理人管理 → 查看申请 → 审核通过
