3.2 企业管理 → 经理人管理
管理员可执行以下操作：

邀请经理人加入

分享二维码、微信链接或直接发送图片

审核加入申请

转移管理员权限（设为企业管理员）

解除经理人与企业的绑定关系

3.3 企业管理 → 司机管理
根据姓名、手机号、邀请人筛选查看

未来将开放企业统一管理所有司机信息的功能

3.4 企业管理 → 佣金分成设置
设置各经理人或方案对应的佣金比例

比例变动后实时生效，无需平台审核

3.5 企业管理 → 佣金结算流程
3.5.1 每月结算流程
每月6号，平台统一生成结算单

企业确认金额后 → 上传发票 → 平台审核

审核通过后 → 法人手机收到对账函短信

法人确认签章后，佣金转账至企业账户

操作建议在PC端进行，使用平台提供的链接打开结算页面

3.5.2 收支流水查看
查看企业所有收入、支出、可提现余额

支持按时间段、金额类型、流水号查询

支持导出Excel文件

3.5.3 推广管理模块
查看每位经理人推广的司机和商家数量

支持按姓名/手机号搜索

查看详细推广数据

可导出报表
