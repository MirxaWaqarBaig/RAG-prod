2.2 雨燕经理人功能介绍
2.2.1 当经理人未绑定企业或未完成企业认证时：
经理人行为： 可以分享租车方案给司机

司机体验： 可以点击查看，但无法下单或签约

解决方案： 建议经理人尽快绑定企业或完成企业认证

2.2.2 经理人加入已有企业
方式一：经理人自主绑定企业
【操作对象：经理人】

路径：
我的 → 绑定企业 → 搜索企业名称 → 提交申请 → 等待审核

审核流程：
企业管理员登录 → 企业管理 → 经理人管理 → 查看申请 → 审核通过

审核后，经理人在“我的”界面显示企业名称，并有“企”字标识

方式二：企业管理员邀请经理人加入
【操作对象：企业管理员】

路径：
我的 → 企业管理 → 经理人管理 → 邀请经理人 →
生成：微信二维码 / 微信链接 / 直接发送微信好友

2.2.3 邀请司机或商家认证加入
【操作对象：经理人】

在小程序中点击“邀请认证”

分享二维码

司机扫码后进入“雨燕租车”小程序注册

商家点击“我是租车公司”，进入PC网页界面（手机端不推荐）

2.2.4 租车流程（细化说明各用户角色）
主要涉及三方：经理人、司机、商家

（1）推荐方案
经理人选择合适的车型与方案

生成分享海报 → 朋友圈/微信群 → 分享给特定司机

（2）司机下单与支付流程
【司机操作】

打开经理人分享链接 → 选择方案 → 提交申请

商家审核通过 → 司机在线签署合同

支付租车费用（可线上或线下）

（3）验车流程
商家选择验车方式：

指派经理人验车（由经理人负责现场验车和签字确认）

或由司机自主验车（司机拍照上传 → 签字 → 经理人复核签字）

如车辆不满意 → 经理人端可发起“换车”

（4）交钥匙
验车成功后，经理人将实体钥匙交付司机

平台记录“已交车”，交易完成

2.3 续租流程
【角色：经理人 & 司机 & 商家】

车快到期 → 经理人联系司机确认是否续租

商家配置续租方案（期限、价格）

司机在小程序内确认续租并支付相关费用

2.4 退车流程
【角色：商家 → 指派经理人验车 → 司机确认退车】

商家发起退车 → 指定退车时间、地址、验车人

验车人（经理人）收到提醒

经理人验车 → 签字 → 提交至司机确认签名

完成清算账单 → 租期结束

2.5 经理人升级成为企业
【角色：经理人】

路径：我的 → 绑定企业 → 升级为企业 → 上传资料 → 交会员费（如有） → 平台审核
升级后自动成为该企业的“企业管理员”，可访问企业管理功能模块

三、雨燕经理人企业功能介绍
适用于认证后的经理人企业账号（企业管理员）

3.1 企业管理 → 业绩查看
查看平台统计的：预计总业绩、实际已到账佣金、所有订单情况

说明：

预计总业绩：所有经理人佣金总额 - 平台抽成（前期平台抽成为 0）

实际总业绩：符合交车15天、无退车条件的订单佣金 - 平台抽成

3.2 企业管理 → 经理人管理
管理员可执行以下操作：

邀请经理人加入

分享二维码、微信链接或直接发送图片

审核加入申请

转移管理员权限（设为企业管理员）

解除经理人与企业的绑定关系

3.3 企业管理 → 司机管理
根据姓名、手机号、邀请人筛选查看

未来将开放企业统一管理所有司机信息的功能

3.4 企业管理 → 佣金分成设置
设置各经理人或方案对应的佣金比例。
