一、雨燕经理人简介
1.1 什么是雨燕租车经理人
雨燕租车经理人是雨燕租车平台专为网约车租赁中介企业打造的销售管理工具。从“获客 → 销售 → 签约 → 验车 → 培训”全流程数字化，便于中介企业提升业务效率，降低人工成本，扩大经营范围，获取更大利润。

1.2 为什么要成为雨燕租车经理人
作为雨燕平台认证的经理人，你将能够：

获得平台提供的各类车辆资源（新能源、油车、网约车等）

在线完成租车闭环：包括推荐方案、签约、支付、交车、退车等

自动获取平台推荐的潜在客户线索，无需高成本寻找司机客户

二、如何成为雨燕租车经理人
2.1 雨燕经理人注册流程
【适用对象：经理人（个人/企业）】
小程序入口方式：
微信搜索：“雨燕租车”小程序

微信扫码：扫描平台二维码直接进入注册页面

注册与认证材料准备：
点击“立即加入”按钮 → 进入实名认证/企业认证页面

根据身份选择：

个人经理人认证：适用于个体，需提交身份证、银行卡等

企业经理人认证：适用于中介公司，需提交法人信息、营业执照等

所需材料（按认证类型区分）：

居民身份证（正反面，清晰）

个体工商户或公司营业执照

银行开户信息（法人或对公账户）

照片示例：所有证件需清晰、有效期内

实名认证流程（个人经理人）：
上传身份证正反面图片

如果是由其他经理人推荐，则填写推荐码（用于佣金绑定）

企业认证流程（企业经理人）：
上传法人身份证（正反面）

选择营业执照类型（企业/个体户）并上传清晰营业执照图片

填写公司账户信息

如果没有对公账户，可填法人个人银行卡账户

如果支行名称搜索不到，可选择所属总行

企业会员费

当前阶段免费开放注册（建议尽快认证）

银联认证流程（经理人企业）：
注：该认证流程适用于企业经理人，需通过银商平台进行实名审核。

所需材料包括：

身份证正反

营业执照（三证合一）

对公账户信息（含银行开户许可证/印章页）

法人及股东资料（身份信息）

雨燕小程序截图、门头照片、店内照片（需加水印）

步骤：

进入“银商商户自助签约平台”（PC推荐）

填写身份信息、商户信息、银行账户信息

完成小额打款验证（确认收到金额）

上传相关图片与门店信息

签署电子协议（法人手机号验证码）

等待银联审核

2.2 雨燕经理人功能介绍
2.2.1 当经理人未绑定企业或未完成企业认证时：
经理人行为： 可以分享租车方案给司机

司机体验： 可以点击查看，但无法下单或签约

解决方案： 建议经理人尽快绑定企业或完成企业认证

2.2.2 经理人加入已有企业
方式一：经理人自主绑定企业
【操作对象：经理人】

路径：
我的 → 绑定企业 → 搜索企业名称 → 提交申请 → 等待审核

审核流程：
企业管理员登录 → 企业管理 → 经理人管理 → 查看申请 → 审核通过

审核后，经理人在“我的”界面显示企业名称，并有“企”字标识

方式二：企业管理员邀请经理人加入
【操作对象：企业管理员】

路径：
我的 → 企业管理 → 经理人管理 → 邀请经理人 →
生成：微信二维码 / 微信链接 / 直接发送微信好友

2.2.3 邀请司机或商家认证加入
【操作对象：经理人】

在小程序中点击“邀请认证”

分享二维码

司机扫码后进入“雨燕租车”小程序注册

商家点击“我是租车公司”，进入PC网页界面（手机端不推荐）

2.2.4 租车流程（细化说明各用户角色）
主要涉及三方：经理人、司机、商家

（1）推荐方案
经理人选择合适的车型与方案

生成分享海报 → 朋友圈/微信群 → 分享给特定司机

（2）司机下单与支付流程
【司机操作】

打开经理人分享链接 → 选择方案 → 提交申请

商家审核通过 → 司机在线签署合同

支付租车费用（可线上或线下）

（3）验车流程
商家选择验车方式：

指派经理人验车（由经理人负责现场验车和签字确认）

或由司机自主验车（司机拍照上传 → 签字 → 经理人复核签字）

如车辆不满意 → 经理人端可发起“换车”

（4）交钥匙
验车成功后，经理人将实体钥匙交付司机

平台记录“已交车”，交易完成

2.3 续租流程
【角色：经理人 & 司机 & 商家】

车快到期 → 经理人联系司机确认是否续租

商家配置续租方案（期限、价格）

司机在小程序内确认续租并支付相关费用

2.4 退车流程
【角色：商家 → 指派经理人验车 → 司机确认退车】

商家发起退车 → 指定退车时间、地址、验车人

验车人（经理人）收到提醒

经理人验车 → 签字 → 提交至司机确认签名

完成清算账单 → 租期结束

2.5 经理人升级成为企业
【角色：经理人】

路径：我的 → 绑定企业 → 升级为企业 → 上传资料 → 交会员费（如有） → 平台审核
升级后自动成为该企业的“企业管理员”，可访问企业管理功能模块

三、雨燕经理人企业功能介绍
适用于认证后的经理人企业账号（企业管理员）

3.1 企业管理 → 业绩查看
查看平台统计的：预计总业绩、实际已到账佣金、所有订单情况

说明：

预计总业绩：所有经理人佣金总额 - 平台抽成（前期平台抽成为 0）

实际总业绩：符合交车15天、无退车条件的订单佣金 - 平台抽成

3.2 企业管理 → 经理人管理
管理员可执行以下操作：

邀请经理人加入

分享二维码、微信链接或直接发送图片

审核加入申请

转移管理员权限（设为企业管理员）

解除经理人与企业的绑定关系

3.3 企业管理 → 司机管理
根据姓名、手机号、邀请人筛选查看

未来将开放企业统一管理所有司机信息的功能

3.4 企业管理 → 佣金分成设置
设置各经理人或方案对应的佣金比例

比例变动后实时生效，无需平台审核

3.5 企业管理 → 佣金结算流程
3.5.1 每月结算流程
每月6号，平台统一生成结算单

企业确认金额后 → 上传发票 → 平台审核

审核通过后 → 法人手机收到对账函短信

法人确认签章后，佣金转账至企业账户

操作建议在PC端进行，使用平台提供的链接打开结算页面

3.5.2 收支流水查看
查看企业所有收入、支出、可提现余额

支持按时间段、金额类型、流水号查询

支持导出Excel文件

3.5.3 推广管理模块
查看每位经理人推广的司机和商家数量

支持按姓名/手机号搜索

查看详细推广数据

可导出报表
